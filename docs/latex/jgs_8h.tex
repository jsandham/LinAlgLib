Jacobi iterative linear solver. 

{\ttfamily jac} solves the sparse linear system $A$ \texorpdfstring{$\ast$}{*} $x$ = $b$ using the Jacobi iterative solver. Consider decomposing the sparse matrix $A$ as\+:

\[
  A := L + U + D,
\] where  \[
  \begin{array}{ll}
      L, & \text{Is the strictly lower part of A} \\
      U, & \text{Is the strictly upper part of A} \\
      D, & \text{Is the diagaonal part of A}
  \end{array}
\] Inserting into $A$ \texorpdfstring{$\ast$}{*} $x$ = $b$ we get\+:  \[
  \begin{array}{ll}
      D * x^{k+1} + (L + U) * x^{k} = b, \\
      x^{k+1} = D^{-1} * (b - (L + U) * x^{k}), &
  \end{array}
\]

\begin{DoxyNote}{Note}
Convergence is only guaranteed if $A$ is strictly diagonally dominant
\end{DoxyNote}
