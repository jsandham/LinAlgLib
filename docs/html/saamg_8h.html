Smoothed Aggregation setup.

saamg_setup generates the hierarchy of restriction, prolongation, and coarse grid operators using Smoothed Aggregation

Parameters
[in]csr_row_ptrarray of m+1 elements that point to the start of every row of the sparse CSR matrix.
[in]csr_col_indarray of nnz elements containing the column indices of the sparse CSR matrix.
[in]csr_valarray of nnz elements containing the values of the sparse CSR matrix.
[in]mnumber of rows in the sparse CSR matrix
[in]nnumber of columns in the sparse CSR matrix
[in]nnznumber of non-zeros in the sparse CSR matrix
[in]max_levelmaximum number of levels in the hierarchy of coarse grids
[out]heirarchystructure holding the heirarchy of restriction, prolongation, and coarse grid operators
Example
std::vector<int> csr_row_ptr;
std::vector<int> csr_col_ind;
std::vector<double> csr_val;
load_mtx_file(matrix_file, csr_row_ptr, csr_col_ind, csr_val);
int m = (int)csr_row_ptr.size() - 1;
int n = m;
int nnz = (int)csr_val.size();
// Solution vector
std::vector<double> x(m, 0.0);
// Righthand side vector
std::vector<double> b(m, 1.0);
heirarchy hierachy;
saamg_setup(csr_row_ptr.data(), csr_col_ind.data(), csr_val.data(), m, m, nnz, 100, hierachy);
amg_solve(hierachy, x.data(), b.data(), 10, 10, 0.00001, Cycle::Vcycle, Smoother::Gauss_Siedel);
