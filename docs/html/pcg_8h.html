Preconditioned conjugate gradient iterative linear solver.

pcg solves the sparse linear system A*x = b using the preconditioned conjugate gradient iterative solver.

Note
Requires the sparse matrix A to be symmetric
Parameters
[in]csr_row_ptrarray of n+1 elements that point to the start of every row of the sparse CSR matrix.
[in]csr_col_indarray of nnz elements containing the column indices of the sparse CSR matrix.
[in]csr_valarray of nnz elements containing the values of the sparse CSR matrix.
[in,out]xarray of n elements containing the solution values of A*x=b
[in]barray of n elements containing the righthad side values of A*x=b.
[in]nsize of the sparse CSR matrix
[in]omegaomega
[in]tolstopping tolerance
[in]max_itermaximum iterations allowed
Return values
numberof iterations actually used in the solver
Example
std::vector<int> csr_row_ptr;
std::vector<int> csr_col_ind;
std::vector<double> csr_val;
load_mtx_file(matrix_file, csr_row_ptr, csr_col_ind, csr_val);
int m = (int)csr_row_ptr.size() - 1;
int n = m;
int nnz = (int)csr_val.size();
// Solution vector
std::vector<double> x(m, 0.0);
// Righthand side vector
std::vector<double> b(m, 1.0);
int it = pcg(csr_row_ptr.data(),
csr_col_ind.data(),
csr_val.data(),
x.data(),
b.data(),
m,
0.00001,
1000);
