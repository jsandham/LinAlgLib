Algebraic Mulitgrid solver.

amg_solve solves a linear system using algebraic multigrid

Parameters
[in]heirarchyStructure holding the heirarchy of restriction, prolongation, and coarse grid operators. Must be filled by calling saamg_setup or rsamg_setup prior to calling amg_solve.
[in,out]xArray of m elements containing the solution values of A*x=b
[in]bArray of m elements containing the righthad side values of A*x=b.
[in]n1Number pre-smoothing steps
[in]n2Number post-smoothing steps
[in]tolTolerance used to determine stopping iteration.
[in]cycleAlgebaric multigrid cycle type. Can be Vcycle, Wcycle, Fcycle.
[in]smootherSmoother type used. Can be Jacobi, Gauss_Siedel, Symm_Gauss_Siedel, SOR, or SSOR.
Example
std::vector<int> csr_row_ptr;
std::vector<int> csr_col_ind;
std::vector<double> csr_val;
load_mtx_file(matrix_file, csr_row_ptr, csr_col_ind, csr_val);
int m = (int)csr_row_ptr.size() - 1;
int n = m;
int nnz = (int)csr_val.size();
// Solution vector
std::vector<double> x(m, 0.0);
// Righthand side vector
std::vector<double> b(m, 1.0);
heirarchy hierachy;
saamg_setup(csr_row_ptr.data(), csr_col_ind.data(), csr_val.data(), m, m, nnz, 100, hierachy);
amg_solve(hierachy, x.data(), b.data(), 10, 10, 0.00001, Cycle::Vcycle, Smoother::Gauss_Siedel);
